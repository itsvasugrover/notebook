# What is Artifactory?

Artifactory is a universal artifact repository manager developed by JFrog. It supports all major packaging formats, build tools, and CI/CD tools.

## Key Features

- Stores and manages binary artifacts
- Supports various package types like Maven, npm, Docker, etc.
- Provides security and access control
- Integrates with CI/CD pipelines
